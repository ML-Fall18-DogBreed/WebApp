Dog Breed Classification Project

Project Overview

The project is to build a tool that can recognize the breed of a dog from an image. This idea originated from a Kaggle challenge which provides the data and specifications for the tool. Although the competition has since ended, it is still a worthwhile challenge with practical applications. The sections below expand on the purpose, prior work and challenges, and plan.
