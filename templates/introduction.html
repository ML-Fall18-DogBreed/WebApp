Introduction

Dog Breed Classification Project



Project Overview

The project is to build a tool that can recognize the breed of a dog from an image. This idea originated from a Kaggle challenge which provides the data and specifications for the tool. Although the competition has since ended, it is still a worthwhile challenge with practical applications. Our goal was to compare and contrast machine learning models for use in this problem. We tried many approaches to classifying the 120 breeds available in the 10,000 images from the Kaggle challenge. In the Analysis and Visualizations tab of this website, you can find our results.
