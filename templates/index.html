What's That Dog?

Dog Breed Classification Project



Welcome!

This website provides information about our group's CS 539: Machine Learning group project. In this project, we created a set of classifiers to detect the breed of a dog from an input image. Our dataset is visible on the Introduction tab of this website. The results are discussed on the Analysis and Visualization page.

Source code: Github

Our project is hosted on our Github team page in a set of two repositories.

In the 'breed-classifier' repository is everything related to the Convolutional Neural Networks implemented in Python. Follow the README for full setup instructions and scroll to the bottom for a detailed chart of most of the network structures that were implemented, with results.

In the repository called 'breed-classifier-SGD', you can find Python scripts for running a variety of other models to varying degrees of accuracy. To validate our test accuracy, clone the repository, download the trained model from this link. Install all required packages including numpy, pandas, sklearn, joblib and matplotlib. Launch breed-classifier-SGD-SVM/DogClassifier_SVM_HOG.py, an accuracy will be reported after fitting 1000 randomly selected images within the dataset and an confusion matrix will be presented. Additionally, the model is stored in a .joblib file at this public Google Drive link. Although we wanted to make it available at this website, it was impossible due to space limitations on Heroku's free hosting service.
